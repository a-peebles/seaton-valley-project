Updated on 17/04/2018 Seaton Valley - Pre-release Version 1 • Min SDK Version: 19 (Android 4.4 - KitKat) • Target SDK Version: 27 (Android 8.1 - Oreo) • Gradle version 4.1 • Android Plugin Version 3.0.1 • To compile the source code, it needs the latest version of Android Studio. Requirements • Requires an Internet and a GPS connection. • A device running Android 4.4 (Android KitKat) or newer, and Google Play services 11.8.0 or higher. • The webserver and the REST API of Seaton Valley Council must be online and functionable. • The webserver of Google Places must be online and functionable. • The Twitter API and the Twitters webserver must online and functionable. • The Firebase cloud server must be functional. • The Google Maps application (for the best functionality) or a web browser installed.
